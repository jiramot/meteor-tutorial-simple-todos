## Table of content

### Install Meteor
```
curl https://install.meteor.com/ | sh
```

### Creating your first app
```
meteor create simple-todos
cd simple-todos
meteor npm install
meteor
```

### Defining views with React components
```
meteor npm install --save react react-dom
```
